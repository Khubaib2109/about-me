# About Me (GitHub Pages)

This repo hosts my personal site via GitHub Pages.

## Live site
https://YOURUSERNAME.github.io/about-me/

## How to publish (GitHub Pages)
1. Repo Settings → Pages
2. Build and deployment:
   - Source: **Deploy from a branch**
   - Branch: **main** (or master) / **root**
3. Save

## Update checklist
- Replace "Your Name" in `index.html`
- Update LinkedIn + GitHub links
- Put your resume at `assets/resume.pdf`
- Set your email in the `data-email` attribute on the “Copy email” button
- Update project links in `script.js`
